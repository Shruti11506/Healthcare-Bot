1. Project Overview

The Healthcare Bot is an AI-powered web application designed to predict possible diseases based on user-input symptoms. The system leverages Machine Learning techniques to analyze symptom data and generate intelligent predictions.

This project demonstrates the practical implementation of Generative AI concepts combined with backend web technologies to create a scalable healthcare assistance system.

2. Problem Statement

Access to preliminary healthcare guidance is limited in many areas. People often ignore early symptoms due to lack of awareness or medical consultation access.

This project aims to provide a smart AI-driven system that can assist users in identifying possible diseases based on symptoms entered.
